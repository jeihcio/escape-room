# Escape Room
Projeto desenvolvido para Arduino Uno

## Instruções
Instalar a biblioteca "keypad.zip" que está na pasta [bibliotecas](/Bibliotecas)

### Instalação
Sketch > Incluir Biblioteca > Adicionar biblioteca .ZIP

## Componentes

- 1	Arduino Uno R3
- 1	Keypad 4x4
- 1	LCD 16 x 2
- 1	Piezo
- 1	LED RGB
- 4	220 Ω Resistor
